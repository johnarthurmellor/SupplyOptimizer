Supply Optimizer

Удобный инструмент для решения задач управления ресурсами

Оптимальная стратегия заказа

  • Рекомендуемое количество товара для заказа составляет шт.
  • Оптимальный средний уровень запаса составляет шт.
  • Рекомендуемый интервал между пополнениями запасов (в днях) составляет дн.
  • Минимальный уровень товара для нового заказа составляет шт.
  • Общие складские издержки за год составят у.е.
